JDBC学习

发表于 2018-03-09 | 更新于 2019-10-16 | 分类于 JavaWeb

一、JDBC（Java DataBase Connectivity）

可以为多种关系型数据库DBMS，提供统一的访问方式，用Java来操作数据库。

1、JDBC API：提供各种操作访问接口，Connection、Statement、PreparedStatement、
ResultSet。
（1）与数据库建立连接
（2）发送SQL语句
（3）返回处理结果
具体是通过以下类/接口实现：
DriverManager：管理JDBC驱动
Connection：连接(通过DriverManager产生)
Statement(PreparedStatement)：增删改查（通过Connection产生）
CallableStatement：调用数据库中的存储过程/存储函数（通过Connection产生）
Result：返回的结果集(通过上面的Statement等产生)

常用方法：
1、Connection产生操作数据库的对象
      Connection产生Statement对象：createStatement()
      Connection产生PreparedStatement对象：PrepareStatement()
      Connection产生CallableStatement对象：PrepareCall()
2、Statement操作数据库：
      增删改：executeUpdate()
      查询：executeQuery()
3、ResultSet:保存结果集 select * from xxx
      next():光标下移，判断是否有下一条数据，返回true/false
      previous():光标上移，返回true/false
      getXXX(字段名/字段名对应的下标位置):获取具体的字段值
4、PreparedStatement操作数据库
    public interface PreparedStatement extends Statement
  增删改：executeUpdate()
  查询:executeQuery()
  *复制操作：setXXX()
5、CallableStatement:调用存储过程、存储函数
   (1)Connection.prepareCall(参数：存储过程或存储函数名)
       参数格式：
           存储过程(无返回值，用Out参数替代)：{call 存储过程名(参数列表)}
           存储函数(有返回值)：{?=call 存储函数名(参数列表)}
   (2)JDBC调用存储过程的步骤：
        a、产生调用存储过程的对象(CallableStatement)  
          cs=connection.prepareCall("...")
        b、通过setXXX()函数处理输入参数值
        c、通过registerOutParameter()函数处理输出参数类型
        d、cs.execute()执行
        e、getXXX()函数接收输出值(返回值)
   (3)JDBC调用存储函数与存储过程的区别在于参数的写法不同，因为存储函数有返回值
6、异常捕获
  （1）ClassNotFoundException
   (2) SQLException
   (3) Exception
7、在finally里关闭相应资源（关闭顺序和打开顺序相反，即先打开的后关闭，后打开的  先关闭），为防止异常可以在关闭之前进行排空验证。

注意：PareparedStatement与Statement在使用时的区别

1、Statement:
     sql
     executeUpdate(sql)
2、PreparedStatement:
     sql(可能存在占位符)
     在创建PreparedStatement对象时，将sql预编译：PrepareStatement(sql);
     executeUpdate()
     setXXX()替换占位符?
3、推荐使用PreparedStatement，原因如下：
  (1)编码更加简便(避免了字符串拼接)
  (2)提高性能（因为有预编译操作，预编译只需要执行一次）
  (3)安全(可以有效的防止SQL注入)
    (SQL注入：将客户输入的内容和开发人员的SQL语句混为一体)

2、JDBC DriverManager:管理不同的数据库驱动。
常见的三种数据库驱动：
驱动jar 具体驱动类
Oracle ojdbc-x.jar oracle.jdbc.OracleDriver
MySQL mysql-connector-java-x.jar com.mysql.jdbc.Driver
SqlServer sqljdbc-x.jar com.microsoft.sqlserver.jdbc.SQLServerDriver
连接字符串
jdbc:oracle:thin:@localhost:1521:ORCL
jdbc:mysql://localhost:3306/数据库实例名
jdbc:microsoft:sqlserver:localhost:1433;databasename=数据库实例名

3、各种数据库驱动：相应的数据库厂商提供的（第三方公司提供），用来连接/直接操作数据库。

二、JDBC学习测试代码

  import java.sql.Statement;
  public class JDBCDemo {
    private static final String URL="jdbc:mysql://localhost:3306/hs";
    private static final String USERNAME="root";
    private static final String PWD="MySQL";

    public static void update() {
        Connection connection=null;
        Statement stmt=null;
        try {
            //1、导入驱动，加载具体的驱动类
                Class.forName("com.mysql.jdbc.Driver");
            //2、与数据库建立连接
                connection = DriverManager.getConnection(URL, USERNAME, PWD);
            //3、发送SQL，执行增删改查
                stmt=connection.createStatement();
            //4、执行SQL语句
                String sql="insert into Student values(4,'hh','男',23,'hh','hh')";
                int count = stmt.executeUpdate(sql); //返回值表示增删改了几条数据
            //5、处理结果
                if(count>0){
                    System.out.println("操作成功！");
                }
            } 
            catch (ClassNotFoundException e1) {
                // TODO Auto-generated catch block
                e1.printStackTrace();
            }catch (SQLException e1) {
                    // TODO Auto-generated catch block
                    e1.printStackTrace();
            }catch(Exception e1){
                e1.printStackTrace();
            }finally{
                //6、关闭连接
                try {
                    if(stmt!=null) stmt.close();
                    if(connection!=null) connection.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
        }
    public static void main(String args[]){
        update();
    }      
}
