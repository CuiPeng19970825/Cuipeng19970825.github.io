一、线性结构—双向链表

发表于 2019-08-23 | 分类于 数据结构与算法

双项链表学习整理

双向链表

单向链表的缺点在于每个节点只能知道自己与下一个节点，无法寻找到上一个节点。而双向链表可以

解决这个问题，通过在节点内部添加一个pre指针，指向它的前一节点。

优点：支持向前遍历，也支持向后遍历；

缺点：在对链表进行增删的时候需要对2个以上指针进行操作。

基类的定义

  • 定义一个节点类DoubleNode

  • insert()方法，插入一个节点

  • pre()方法，获取上一个节点

  • next()方法，获取下一个节点

  • getData()方法，获取节点中的数据

代码实现

package demoDoubleNode;
       public class DoubleNode {
           //上一个节点
           private DoubleNode pre;
           //下一个节点
           private DoubleNode next;
           //数据
           private int data;

           public DoubleNode(int data){
               pre=this;
               next=this;
               this.data=data;
           }

           //插入一个节点
           public void insert(DoubleNode newNode){
               //当前节点
               DoubleNode currentNode=this;
               //当前节点的下一个节点之后作为下下一个节点
               DoubleNode NextNode=this.next;
               //令将当前节点的下一个节点指向新节点
               currentNode.next=newNode;
               //将新节点的前一个节点指向当前节点
               newNode.pre=currentNode;
               //将新节点的下一个节点指向下下一个节点
               newNode.next=NextNode;
               //将下下一个节点的前一个节点指向新节点
               NextNode.pre=newNode;
           }

           //获取上一个节点
           public DoubleNode pre(){
               return this.pre;
           }

           //获取下一个节点
           public DoubleNode next(){
               return this.next;
           }

           //获取数据
           public int getData(){
               return this.data;
           }

       }

测试代码

package demoDoubleNode;
public class TestDoubleNode {
       public static void main(String[] args) {
           DoubleNode n1=new DoubleNode(1);
           DoubleNode n2=new DoubleNode(2);
           DoubleNode n3=new DoubleNode(3);
           DoubleNode n4=new DoubleNode(4);
           System.out.println(n1.pre().getData());
           System.out.println(n1.next().getData());
           n1.insert(n2);
           n2.insert(n3);
           n3.insert(n4);
           System.out.println(n1.pre().getData());
           System.out.println(n4.next().getData());
           System.out.println(n2.next().getData());
       }
   }
