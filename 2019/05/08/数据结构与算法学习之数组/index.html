一、线性结构—数组

发表于 2019-05-08 | 更新于 2019-08-23 | 分类于 数据结构与算法

数组学习整理

数组

数据结构中最基本的一个结构就是线性结构，而线性结构又分为连续存储结构和离散存储结构，

所谓的连续存储结构其实就是数组。

优点 ：

  1. 按照索引查询元素速度快

  2. 按照索引遍历数组方便

缺点：

  1. 数组的大小固定后就不能扩容了

  2. 数组只能存储一种类型的数据

  3. 添加，删除的操作慢，因为要移动其他的元素

适用场景：

频繁查询，对存储空间要求不大，很少增加和删除的情况

基类的定义

  • 定义一个数组类Arr

  • size()方法，获取数组长度

  • add()方法，向数组末尾增加一个元素

  • show()方法，显示数组中的所有元素

  • delete()方法，删除数组中某个位置的元素

  • insert()方法，向数组中插入一个元素

  • get()方法，获取数组中目标位置的元素

  • set()方法，替换数组中目标位置的元素值

  • search()线性查找方法

  • BinarySearch()二分法查找，条件：查找的数组有序

代码实现

package demoArray;
import java.util.Arrays;
public class Arr {
    private int[] arr;

    public Arr (){
        arr=new int[0];

    }

    //获取数组长度
    public int size(){
        return arr.length;
    }

    //向数组末尾增加一个元素
    public void add(int element){
        //新创建一个数组
        int[] newarr=new int[arr.length+1];
        //将原数组中的元素复制到新数组中
        for(int i=0;i<arr.length;i++){
            newarr[i]=arr[i];
        }
        //将新增加的元素放入新数组中
        newarr[arr.length]=element;
        //新数组替换原数组
        arr=newarr;
    }

    //显示数组中的所有元素
    public void show(){
        System.out.println(Arrays.toString(arr));
    }

    //删除数组中某个位置的元素
    public void delete(int index){
        //创建一个新数组
        int[] newarr=new int[arr.length-1];
        //判断目标位置是否越界
        if(index<0||index>arr.length-1){
            throw new RuntimeException("下标越界");
        }
        for(int i=0;i<newarr.length;i++){
            //将删除元素前的元素复制到新数组中
            if(i<index){
            newarr[i]=arr[i];
            //将删除元素后的元素复制到新数组中
            }else{
                newarr[i]=arr[i+1];
            }
        }
        //新数组替换原数组
        arr=newarr;
    }

    //向数组中插入一个元素
    public void insert(int index,int element){
        //创建一个新数组
        int[] newarr=new int[arr.length+1];
        //判断下标是否越界
        if(index<0||index>newarr.length-1){
            throw new RuntimeException("下标越界");
        }
        for(int i=0;i<arr.length;i++){
            //将插入元素前的元素复制到新数组
            if(i<index){
                newarr[i]=arr[i];
            //将插入元素后的元素复制到新数组
            }else{
                newarr[i+1]=arr[i];
            }
        }
        //将插入元素放入到新数组中
        newarr[index]=element;
        //新数组替换原数组
        arr=newarr;
    }

    //获取数组中目标位置元素
    public int get(int index){
        //判断目标位置是否越界
        if(index<0||index>arr.length-1){
            throw new RuntimeException("下标越界");
        }
        return arr[index];
    }

    //替换数组中目标位置元素的值
    public void set(int index,int element){
        //判断目标位置是否越界
        if(index<0||index>arr.length-1){
            throw new RuntimeException("下标越界");
        }
        arr[index]=element;
    }

    //线性查找
    public int search(int target) {
        for(int i=0;i<arr.length;i++){
            if(arr[i]==target){
                return i;
            }
        }
        return -1;
    }

    //二分法查找
    public int BinarySearch(int target){
        // 记录开始位置
        int begin = 0;
        // 记录结束位置
        int end = arr.length - 1;
        // 记录中间的位置
        int mid = (begin + end) / 2;
        // 循环查找
        while (true) {
            //判断什么情况下没有目标元素
            if(begin>=end){
                return -1;
            }
            // 判断中间的这个元素是否是目标元素
            if (arr[mid] == target) {
                return mid;
            } else {
                if (arr[mid] > target) {
                    end = mid - 1;
                } else {
                    begin = mid + 1;
                }
                mid = (begin + end) / 2;
            }
        }
    }
 }

测试代码

package demoArray;
public class TestArr {
    public static void main(String[] args) {
        //创建一个数组对象
        Arr arr=new Arr();
        //获取数组长度
        int size=arr.size();
        System.out.println(size);
        //向数组中增加元素
        arr.add(99);
        arr.add(98);
        arr.add(97);
        System.out.println(arr.size());
        //显示数组中的所有元素
        arr.show();
        //删除数组中第一个位置的元素
        arr.delete(1);
        arr.show();
        arr.add(96);
        arr.add(95);
        arr.add(94);
        arr.add(93);
        arr.show();
        //向数组中第一个位置插入元素98
        arr.insert(1,98);
        arr.show();
        //取出数组第四个位置的元素
        System.out.println(arr.get(4));
        //设置数组中第六个位置元素的值为90
        arr.set(6,90);
        arr.show();
    }
}
