发表于 2019-05-22 | 更新于 2019-08-26 | 分类于 数据结构与算法

二叉树学习整理

二叉树的基本概念

一棵二叉树是节点的一个有限集合，该集合或者为空，或者由一个根节点加上两棵左子树和右子树

组成，每个节点最多有两棵子树，即二叉树不存在度大于2的节点，二叉树的子树有左右之分，其子

树的次序不能颠倒。

满二叉树

  1. 叶子节点都在最后一层

  2. 节点总数：2^n-1(n为树高度)

完全二叉树

  1. 所有叶子节点都在最后一层或倒数第二层

  2. 最后一层的叶子节点在左边连续，倒数第二层的叶子节点在右边连续

二叉树的性质

  1. 若规定根节点的层数为0，则一颗非空二叉树的第i层上最多有2^i(i>=0)个节点;

  2. 若规定只有根节点的二叉树的深度为0，则深度为k的二叉树的最大节点数是2^(k+1)-1(k>=-1)

个;

  1. 对于一颗非空的二叉树，如果叶节点个数为n0,度为2的节点数为n2，则有n0=n2+1;

  2. 具有n个节点的完全二叉树的深度k为[log2n]+1

  3. 对于具有n个节点的完全二叉树，如果按照从上至下和从左至右的顺序对所有节点从0开始顺序编

号，则对于序号为i(0 <= i< n)的节点，有：

5.1. 如果i>0,则序号为i节点的双亲节点的序号为(i-1)/2;如果i=0,则序号为i的节点为根节

点，无双亲节点。

5.2. 如果2i+1<n,则序号为i节点的左孩子节点的序号为2i+1;如果2i+1>=n,则序号为i的节

点无左孩子。

5.3. 如果2i+2<n,则序号为i节点的右孩子节点的序号为2i+2;如果2i+2>=n,则序号为i的节

点无右孩子。
