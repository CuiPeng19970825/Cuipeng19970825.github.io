一、线性结构—单链表

发表于 2019-05-11 | 更新于 2019-08-23 | 分类于 数据结构与算法

单链表学习整理

单链表

通过一个链子把多个结点（元素）连接起来，由数据和地址组成的一个元素，节点本身必须有一个

地址值（就是下一个元素的地址值）链表的每个结点中只包含一个指针域，叫做单链表（即构成链

表的每个节点只有一个指向直接后继结点的指针，且尾节点的指针为null）。

优点：

  1. 链表是很常用的一种数据结构，不需要初始化容量，可以任意加减元素；

  2. 添加或者删除元素时只需要改变前后两个元素结点的指针域指向地址即可，所以添加，

删除很快；

缺点：

  1. 因为含有大量的指针域，占用空间较大；

  2. 查找元素需要遍历链表来查找，非常耗时。

适用场景：

数据量较小，需要频繁增加，删除操作的场景。

基类的定义

  • 定义一个节点类Node

  • append()方法,为当前节点追加节点

  • Next()方法，获取下一个节点

  • getData()方法，获取节点中的数据

  • isLast()方法，判断当前节点是否是最后一个节点

  • remove()方法，删除下一个节点

  • show()方法，显示所有节点信息

  • insert()方法，插入一个节点

代码实现

package demoNode;
public class Node {
       //数据
       private int data;
       //下一个节点
       private Node next;

       //构造方法
       public Node(int data){
           this.data=data;
       }

       //为当前节点追加节点
       public Node append(Node node){
           //当前节点
           Node currentNode=this;
           //循环查找
           while(true){
               //取出下一个节点
               Node nextNode=currentNode.next;
               //如果下一个节点为空，当前节点已经是最后一个节点
               if(nextNode==null){
                   break;
               }
               //赋值给当前节点
               currentNode=nextNode;
           }
           //把需要追加的节点追加到找到的当前节点的下一个节点
           currentNode.next=node;
           return this;
       }

       //获取下一个节点
       public Node Next(){
           return this.next;
       }

       //获取节点中的数据
       public int getData(){
           return this.data;
       }

       //判断当前节点是否是最后一个节点
       public boolean isLast(){
           return next==null;
       }

       //删除下一个节点
       public void remove(){
           //取出下一个节点
           Node newNext=next.next;
           //把下下个节点设置为当前节点的下一个节点
           this.next=newNext;

       }

       //显示所有节点信息
       public void show(){
           Node currentNode=this;
           while(true){
               System.out.print(currentNode.data+" ");
               //取出下一个节点
               currentNode=currentNode.next;
               if(currentNode==null){
                   break;
               }
           }
           System.out.println();
       }

       //插入一个节点
       public void insert(Node newNode){
           Node currentNode=this;
           newNode.next=currentNode.next;
           currentNode.next=newNode;
       }

   }

测试代码

 package demoNode;
 public class TestNode {
    public static void main(String[] args) {
        //创建节点对象
        Node n1=new Node(1);
        Node n2=new Node(2);
        Node n3=new Node(3);
        //用追加节点方法将相应的节点连成单链表
        n1.append(n2).append(n3).append(new Node(4));
        //取出下一个节点中的数据
        System.out.println(n1.Next().Next().getData());
        System.out.println(n1.isLast());
        //显示所有节点信息
        n1.show();
        //删除某个节点的下一个节点
        n1.remove();
        n1.show();
        //插入节点
        n1.insert(n2);
        //追加新节点
        n1.append(new Node(5));
        n1.show();
        System.out.println(n2.getData());
    }
}
